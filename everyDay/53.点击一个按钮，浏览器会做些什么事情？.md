# Problem: 点击一个按钮，浏览器会做些什么事情【呈现效果时流程】？(js部分)

*[interview]: start

1. 点击按钮后创建一个Event实例
2. 然后把事件放到事件队列中，让它等候处理
3. Event循坏线程处理这个事件
4. 沿着DOM路径找到触发事件的元素
5. 如果这个元素上有处理这个事件的默认行为，并且要在DOM事件阶段周期**之前执行**，就执行它的默认行为
6. 捕获阶段
7. 目标阶段
8. 冒泡阶段
9. 如果这个元素上有处理这个事件的默认行为，并且要在DOM事件处理阶段周期**之后执行**，就执行它的默认行为


*[interview]: end
